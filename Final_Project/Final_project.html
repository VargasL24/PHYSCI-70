Final Project

Pitch

My idea for my final project is to make a rocket stable durring flight using MEMS-based rocket guidence systems. What this means is that a small anolog gyroscope and accelerometer would go inside of a small model rocket and make sure it flys straight and steady. The way I plan on making this work is by using a MPU 6050 as the gyroscope and accelerometer, and send that data to an arduino nano.
