Week 10: Machine Building and End Effectors

For this week we had to design a machine using stepper motors. I thought it was going to be easy because of working with servo motors before, however this proved to be false due to the dificult wiring. When I was connecting the motor to the driver using the tutorial, I found it hard to understand the wiring at first however after looking at all of the resources provided in the tutorial I figured it out and got the rest down fairly quickly. While I was wiring a wire wasnt in all of the way and as a result something strange happened. The motor would spin and then stop once a force was applied to it, and then repeat infinitly. Sadly there is no video of this happening because my phone was dead while I was at the lab.


  #include AccelStepper.h

    const int stepPin = 13;  // blue
    const int dirPin = 12;   // orange
    
    // Define a stepper and the pins it will use
    AccelStepper stepper(1, stepPin, dirPin); // initialise accelstepper for a two wire board
    
    void setup() {
      // put your setup code here, to run once:
      pinMode(stepPin, OUTPUT);
      pinMode(dirPin, OUTPUT);
    
      digitalWrite(dirPin, LOW);
    }
    
    void loop() {
      // put your main code here, to run repeatedly:
      digitalWrite(stepPin, LOW);
      delay(100);
      digitalWrite(stepPin, HIGH);
      delay(5);
    }

The way that this works is very simple. There is a off set gear attatched to the stepper motor to spin but only allow two gears to overlap at certain times. Due to the fact I can control the speed of the stepper to make the box open more often or less often.
